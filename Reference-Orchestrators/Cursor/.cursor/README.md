# .cursor drop-in for minimal FITPAC

**Optional.** Copy this folder’s contents into your project’s **.cursor** directory so Cursor has the minimal FITPAC skill.

- Copy **src/cursor/skills/** (or **cursor/skills/** after you drop the package) into **.cursor/skills/**.
- The **minimal-fitpac** skill lets the user explicitly say "run FITPAC" or "minimal FITPAC" to trigger the full loop.

If you only use the **.cursorrules** from the package, the loop runs when the user types their idea in Chat; the skill is an alternative explicit trigger.
